### V-Hud:
A work-in-progress user interface overhaul, for Grand Theft Auto: San Andreas, based on Grand Theft Auto: V.\
Project has been made in order to replicate accurately most of the UI elements present in GTA:V.

### Compatibility:
Currently compatible only with GTA: San Andreas 1.0 US.

### Content and features:
* Main and Pause Menu.
* HUD
* Radar (2D and 3D)
* Help text variation
* Subtitles
* Blips and Markers
* Weapon Wheel
* Radio Wheel [WIP]
* Police system
* GPS
* Timebars
* Extended crosshairs
* Player Statistics screen
* Wasted/Busted screen
* Mission complete screen [WIP]
* High quality textures
* Overlay layers (black and white effects and more)
* Menu panels (shop and more) [WIP]
* Cellphone [WIP]
* Detailed menu map and legend
* Fully working Gallery to view photos taken with the camera
* Easy to use API for new plugins development [WIP]
